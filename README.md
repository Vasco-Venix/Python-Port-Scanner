# Python-Port-Scanner
Ferramenta de Port Scanning desenvolvida em Python que utiliza multithreading para analisar portas TCP de forma rápida e eficiente. Permite definir um intervalo de portas, identifica as que estão abertas e apresenta o serviço associado quando disponível.

## Funcionalidades
- Scan de portas TCP.
- Suporte para intervalo de portas personalizado.
- Multithreading para maior desempenho.
- Identificação do serviço associado à porta (quando disponível).
- Barra de progresso durante o scan.
